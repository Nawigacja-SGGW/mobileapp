# NawigacjaSGGW-mobile

## How to run

```bash
npm install
# android
npm run android:go
# android
npm run ios:go

```

## Direcotry structure

- app - screens of the app, file based routing.
- components - components used in the app.
- translation - translations of the app, i18n and stuff.
- hooks - react hooks.
- store - zustand stores.
- core - some important libraries.
